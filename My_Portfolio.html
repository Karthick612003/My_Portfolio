About Me

I'm an Electrical & Electronics Engineering graduate (2025) currently working as an Application Engineer at Global Industrial Garment Machines (GIGM), Bengaluru. My daily work involves control panel wiring, schematic tracing, digital/analog I/O testing, and configuring Nidec S100/C200 Variable Frequency Drives.

I'm actively transitioning into Industrial Automation Design and Industry 4.0 — learning PLC logic design on Rockwell platforms, building IIoT pipelines with MQTT and Modbus, and developing Python tools for machine health monitoring.

My most personal project is a solar-powered, LoRa-based remote irrigation system I'm building for my family's farm in Tamil Nadu — a real-world IIoT capstone that combines ESP32, LoRa, MQTT, and Python into one complete pipeline.

I document everything on GitHub under PLC_Programming and Automation_Learning_Lab — because learning in public is the best way to grow.
